# Todo-App

It is a simple application in which users can enter thir to-do's and check if completed and/or delete their to-do(s).

## About the app:
It has the following features:
*Add task(Using the floating action button)
*Check the completed tasks from the checkbox and it automatically strikes out the task.
*To delete a task simply long press it.
*It also shows the number of tasks.
